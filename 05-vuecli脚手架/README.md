### 需先安装node和全局webpack
### npm install -g @vue/cli         全局安装，脚手架3
### npm install -g @vue/cli-init    全局安装，脚手架2
### vue --version                   查看版本

### vue init webpack my-project     vue cli2初始化项目
#### vue-router 路由
#### ESLint js代码规范，在编译时报错
#### unit tests 单元测试
#### e2e tests 端到端测试

### vue create my-project           vue cli3初始化项目
